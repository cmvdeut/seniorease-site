# üé® Facebook Coverfoto Maken met Canva - Stap voor Stap

## ‚ö° Snelste Methode (5 minuten)

### **Stap 1: Ga naar Canva**
1. Open [Canva.com](https://www.canva.com)
2. Maak een gratis account (als je die nog niet hebt)
3. Klik op "Ontwerp maken"

### **Stap 2: Kies Facebook Cover**
1. Zoek naar "Facebook Cover"
2. Klik op "Facebook Cover" (1640 x 859 pixels)
3. Of maak custom size: 1640 x 859 pixels

### **Stap 3: Achtergrond Instellen**
1. Klik op "Achtergrond" in het linker menu
2. Kies "Kleur"
3. Voer in: `#F5EEE6` (cream kleur)
4. Of gebruik gradient:
   - Klik op "Gradient"
   - Kies "Aangepast"
   - Kleur 1: `#F5EEE6`
   - Kleur 2: `#E0D5CA`

### **Stap 4: Logo Toevoegen**
1. Klik op "Uploads" in linker menu
2. Upload je `heart-logo.png` bestand
3. Sleep het logo naar het canvas
4. Plaats het bovenaan in het midden
5. Grootte: ongeveer 180x180 pixels
6. **Optioneel:** Voeg witte cirkel toe:
   - Klik op "Vormen" ‚Üí "Cirkel"
   - Maak cirkel 200x200 pixels
   - Kleur: Wit (#FFFFFF)
   - Plaats achter logo
   - Border: 6px, kleur `#8B5E3C`

### **Stap 5: Titel Toevoegen**
1. Klik op "Tekst" ‚Üí "Voeg een koptekst toe"
2. Typ: **"SeniorEase"**
3. **Font:** Bold, groot (80-90px)
4. **Kleur:** `#8B5E3C` (bruin)
5. Plaats onder het logo, gecentreerd

### **Stap 6: Tagline Toevoegen**
1. Klik op "Tekst" ‚Üí "Voeg een ondertitel toe"
2. Typ: **"Handige technologie zonder gedoe"**
3. **Font:** Medium, middelgroot (40-45px)
4. **Kleur:** `#4A4A4A` (donkergrijs)
5. Plaats onder de titel, gecentreerd

### **Stap 7: Website Toevoegen**
1. Klik op "Tekst" ‚Üí "Voeg een tekstvak toe"
2. Typ: **"seniorease.nl"**
3. **Font:** Bold, groot (35-40px)
4. **Kleur:** `#8B5E3C` (bruin)
5. Plaats onderaan, gecentreerd

### **Stap 8: Downloaden**
1. Klik op "Downloaden" (rechtsboven)
2. Kies "PNG" of "JPG"
3. Kwaliteit: "Hoog"
4. Download het bestand

### **Stap 9: Uploaden naar Facebook**
1. Ga naar je Facebook pagina
2. Klik op "Foto toevoegen" op de coverfoto plek
3. Upload het gedownloade bestand
4. Pas eventueel aan (crop)
5. Klik "Opslaan"
