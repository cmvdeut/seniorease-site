## ğŸ”§ Problemen Oplossen

### **Animatie start niet:**
- Controleer of JavaScript is ingeschakeld
- Probeer een andere browser (Chrome, Firefox)
- Refresh de pagina (F5)

### **Cursor beweegt niet goed:**
- Zet browser op fullscreen
- Controleer of browser niet is gezoomd (Ctrl+0)
- Gebruik 1920x1080 resolutie

### **Recording kwaliteit laag:**
- Verhoog bitrate in recording software
- Gebruik 1920x1080 resolutie
- Zet browser op fullscreen
