## üé¨ Stap-voor-Stap Instructies

### **Stap 1: Open de Animatie**
1. Open `index.html` in je browser
2. Klik op "Mijn Bibliotheek" (of welke animatie je wilt opnemen)
3. De animatie opent in een nieuw tabblad

### **Stap 2: Zet Browser op Fullscreen**
- Druk op **F11** (Windows) of **Cmd+Ctrl+F** (Mac)
- Dit geeft de beste kwaliteit voor opname

### **Stap 3: Start Screen Recording**

#### **Optie A: OBS Studio (Aanbevolen - Gratis)**
1. **Download OBS Studio:** https://obsproject.com/
2. **Installeer** en open OBS
3. **Maak nieuwe scene:**
   - Klik op "+" bij "Scenes"
   - Noem het "SeniorEase Animatie"
4. **Voeg bron toe:**
   - Klik op "+" bij "Sources"
   - Kies "Window Capture" of "Display Capture"
   - Selecteer je browser venster
5. **Stel resolutie in:**
   - **Voor normale video's (16:9):**
     - Settings ‚Üí Video
     - Base Canvas: 1920x1080
     - Output: 1920x1080
   - **Voor YouTube Shorts (9:16):**
     - Settings ‚Üí Video
     - Base Canvas: 1080x1920 (vertical!)
     - Output: 1080x1920
6. **Start opname:**
   - Klik op "Start Recording"
   - Wacht tot de animatie klaar is
   - Klik op "Stop Recording"

#### **Optie B: Windows Game Bar (Windows 10/11)**
1. Druk op **Win + G**
2. Klik op de record knop (rode cirkel)
3. Of druk op **Win + Alt + R** om direct te starten
4. Druk opnieuw op **Win + Alt + R** om te stoppen
5. Video staat in: `C:\Users\[jouw naam]\Videos\Captures\`

#### **Optie C: QuickTime (Mac)**
1. Open QuickTime Player
2. File ‚Üí New Screen Recording
3. Klik op de record knop
4. Selecteer het browser venster
5. Klik op "Stop" wanneer klaar

#### **Optie D: Browser Extensie (Eenvoudig)**
1. Installeer "Loom" of "Screencastify" extensie
2. Klik op de extensie icoon
3. Kies "Record Screen"
4. Selecteer browser tab
5. Start opname

### **Stap 4: Wacht Tot Animatie Klaar Is**
- De animatie loopt automatisch door alle stappen
- Wacht tot het einde (success scherm)
- Stop dan de recording

### **Stap 5: Bewerk (Optioneel)**
1. **Open video** in editing software:
   - Windows: Video Editor (ingebouwd)
   - Mac: iMovie (gratis)
   - Of: DaVinci Resolve (gratis, professioneel)
2. **Voeg toe (optioneel):**
   - Intro met SeniorEase logo
   - Voice-over (spreek de stappen uit)
   - Outro met abonneer knop
   - Tekst overlays (grote tekst voor belangrijke punten)

### **Stap 6: Exporteer**
- **Formaat:** MP4
- **Resolutie:** 1920x1080 (Full HD)
- **Codec:** H.264
- **Kwaliteit:** Hoog

### **Stap 7: Upload naar YouTube**
1. Ga naar YouTube Studio
2. Klik op "Create" ‚Üí "Upload video"
3. Sleep je video bestand
4. Voeg titel, beschrijving en tags toe
5. Maak een thumbnail
6. Publiceer!

## ‚ö†Ô∏è Veelvoorkomende Problemen

### **Probleem: Video is te donker**
- **Oplossing:** Verhoog helderheid in browser of gebruik Display Capture in plaats van Window Capture

### **Probleem: Cursor is niet zichtbaar**
- **Oplossing:** Cursor zou automatisch zichtbaar moeten zijn. Controleer of browser op fullscreen staat

### **Probleem: Video is wazig**
- **Oplossing:** Verhoog bitrate in recording software (minimaal 5000 kbps)

### **Probleem: Audio ontbreekt**
- **Oplossing:** Controleer audio instellingen in recording software. Voice-over voeg je later toe in editing software.
